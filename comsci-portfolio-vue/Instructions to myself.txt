To update the website, run in the project folder (comsci-portfolio-vue):
npm run build
firebase deploy


Local server: npm run dev


If in powershell script is prevented from running, use this to bypass the permission (or use command line):
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass
